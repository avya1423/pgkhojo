# How to replace the SheherSaathi code on GitHub

Use this guide if you are editing the project directly from the GitHub website and want to copy-paste the updated code into your existing repository.

## Option A: Replace files from the GitHub website

1. Open your GitHub repository: `avya1423/SheherSaathi`.
2. Click the file you want to update, for example `index.html`.
3. Click the pencil icon **Edit this file**.
4. Press `Ctrl + A` / `Cmd + A` to select the old code.
5. Paste the new code for that same file.
6. Click **Commit changes...**.
7. Use a commit message like `Update SheherSaathi UI`.
8. Choose **Commit directly to the main branch** if you want the live GitHub Pages site to update immediately.
9. Repeat the same steps for every changed file.

## Files changed in this update

Copy and replace these files in GitHub:

- `index.html`
- `style.css`
- `script.js`

No image upload is required for this update because the new design uses CSS and existing remote images.

## Option B: Upload files from your computer

1. Download or copy the updated project files to your computer.
2. Open the GitHub repository.
3. Click **Add file**.
4. Click **Upload files**.
5. Drag the updated `index.html`, `style.css`, and `script.js` files into GitHub.
6. Confirm that GitHub says the files will be replaced.
7. Click **Commit changes**.

## After replacing files

GitHub Pages usually updates within a few minutes.

Check the live site here:

`https://avya1423.github.io/SheherSaathi/`

If the old design still appears, hard refresh the browser:

- Windows/Linux: `Ctrl + F5`
- Mac: `Cmd + Shift + R`

## Important notes

- Do not paste all files into one file. Each file must be replaced separately.
- `index.html` code goes only inside `index.html`.
- `style.css` code goes only inside `style.css`.
- `script.js` code goes only inside `script.js`.
- Keep `data.js` as it is unless you want to change PG listing data.
